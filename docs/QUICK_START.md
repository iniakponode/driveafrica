# Safe Drive Africa Quick Start

## 1. Install and launch
Install the app and open it on an Android 8.0+ device.

## 2. Permissions
Allow:
- Location (fine/coarse)
- Activity recognition
- Foreground service
- Notifications (Android 13+)

## 3. Register or log in
- New users: choose a fleet option, enter email + password, tap Register.
- Returning users: enter email + password, tap Log in.

Offline note:
- First login on a fresh device needs internet.
- After a successful online login, you can log in offline with cached credentials.

## 4. Start recording trips
Open Record Trip:
- Tap Start Trip when driving.
- Tap End Trip when finished.

Enable Auto Trip Detection in Settings if you want automatic trip start/stop.

## 5. Reports and tips
- Open Reports, select a period, tap Generate Report.
- Review driving tips on the Home screen.

## 6. Daily alcohol questionnaire
Complete it once per day from the Home reminder or quick action.

## 7. Settings
Use Settings to:
- Toggle Auto Trip Detection
- Change sensitivity
- Allow metered uploads
- Trigger Manual Sync
- Copy your Driver Profile ID for support requests
